Mixto: móvil y de escritorio

El sistema de red Bootstrap 3 tiene cuatro niveles de clases: xs (teléfonos), SM (comprimidos), MD (equipos de escritorio), y LG (de escritorio más grandes). Usted puede utilizar casi cualquier combinación de estas clases para crear diseños más dinámicos y flexibles.

Cada nivel de las clases de escalas de arriba, es decir, si usted planea en la fijación de los mismos anchos para xs y sm, sólo es necesario especificar xs.
